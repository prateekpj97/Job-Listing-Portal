  Post jobs
  Apply for jobs
  Forms + Filtering

4.  Task Queue: Celery + Redis

6.  Frontend: Bootstrap 5, jQuery
7.  Testing: Pytest
8.  Deployment: Docker & Docker Compose
9.  Add unit tests
